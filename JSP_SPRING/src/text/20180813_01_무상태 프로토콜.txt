 + Stateless Protocol
  - 무상태 프로토콜

  - HTTP

  - HTTP는 연결 시 하나의 요청-응답으로 처리되는 Stateless프로토콜
   ( 연속된 요청을 연결 짓지 못한다 )
   ( 동일한 사용자가 계속 요청을 보내는 지 판단하지 못함 )




 + 상태 관리 매커니즘
  - 웹서버가 사용자의 상태 정보를 관리하는 매커니즘



  - 쿠키, Cookie
	서버가 특정 정보를 클라이언트에 저장하는 기술


  - 세션, Session
	서버-클라이언트 간 상호 정보 교환 기술
