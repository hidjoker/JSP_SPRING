
 + 직렬화, Serialization
  - 스트림 통과를 위해 기존의 데이터를 1바이트 단위로 쪼개는 것

 + 역직렬화, Deserialzation
  - 스트림을 통과해 전달된 데이터(1바이트단위)를 결합하는 것


 + 마샬링, Marshalling
  - 데이터를 전달하기 위해 상대방의 표기법으로 변환하는 것
  - 주로 JSON 형식을 활용함
  - XML을 이용하기도 함

 + 언마샬링, Unmarshalling
  - JSON형식(XML형식)으로 전달된 데이터를 내가 사용하는 타입으로
   변환하는 것
